# ADT^A01 Mapping

## Scope

Maps `ADT^A01` (Admit/Visit Notification) messages to FHIR R4 resources:
- `Patient`
- `Encounter`

## Segment Mapping Summary

| HL7v2 Segment | FHIR Resource | Notes |
|---------------|---------------|-------|
| `PID`         | `Patient`     | Patient identity fields |
| `PV1`         | `Encounter`   | Visit/encounter context |
